Minyatur – Thumbnail Module Variants

This demo showcases the different thumbnail styles available in the Minyatur slider component. You can switch the thumbnail type using the data-minyatur-thumbnail-type attribute or JS configuration.

1. Dot Thumbnails (configured via data-attributes)

Dots appear below the slider to indicate each slide. Configured entirely via HTML.

2. Slider Thumbnails (configured via JS)

A mini slider strip showing slide previews, scrollable if overflowed.

3. Basic Thumbnails (configured via JS)

Static thumbnail images listed below the main slider without interaction effects.
